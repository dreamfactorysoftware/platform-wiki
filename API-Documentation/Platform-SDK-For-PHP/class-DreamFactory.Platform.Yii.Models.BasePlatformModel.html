Class BasePlatformModel

BasePlatformModel.php

Defines two "built-in" behaviors: DataFormat and TimeStamp - DataFormat automatically formats date/time values for the target database platform (MySQL, Oracle, etc.) - TimeStamp automatically updates create_date and lmod_date columns in tables upon save.
