<p>The <b>Data</b> menu item in the left-side menu opens the <b>Manage Data</b> module, where the administrator can select tables in the  data editor view, and edit the data.</p>

<p><b>NOTE:</b> From the <b>Manage Schema</b> module, the administrator can edit, delete, sort, search, and filter application data in the database tables.</p>

<p>To enter a new row of data, enter the data name in the Name column in the first row of the table, and click the Save icon to the far left.</p>

<p>To edit a field, click on the value you would like to change, enter the new value, and click the Save icon.</p>

<p>To delete a row click the red Delete icon (X) on the row you would like to delete.</p>

<p>To access the Search and Group data feature, click the drop-down arrow in the right end of the header row. To search data begin typing in the search box. To group data, click the icon to the right of the column name fields. Multiple groupings can be applied. The order of the group using a number to the right of the group button. The checkbox to the left of the column names in the Search dropdown can be used to hide and show fields.</p>
